Solution

My ionic2 lib was called alphacore and before publishing it I was using npm link to consume it from my application. Well, it seems that ionic2 build is not handling properly symlinks. I guess it’s ngc (angular compiler). So the solution to that problem was:

  1. Unlink the node_module
  1. Install properly my library

or copy the library directly into the node_modules folder.

It took me a while to find it! Hope this doesn’t happen to anyone!
